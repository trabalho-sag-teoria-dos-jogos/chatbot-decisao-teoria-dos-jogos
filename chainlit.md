# Consultor de Estratégia Competitiva

Sistema de apoio à decisão gerencial baseado em **Teoria dos Jogos**,
desenvolvido para a disciplina de Sistemas de Apoio à Gestão.

## O que este sistema faz

Dado um concorrente específico, o sistema ajuda você a decidir **qual
estratégia competitiva adotar**, formalizando a decisão como um jogo
estratégico e aplicando os conceitos de **estratégia dominante** e
**equilíbrio de Nash**.

## O que este sistema não faz

Não avalia viabilidade financeira, operacional ou de mercado do negócio
como um todo — apenas o aspecto estratégico/competitivo frente a um
concorrente.

## Como começar

Envie, no chat, o link do site de um concorrente que você quer analisar.
